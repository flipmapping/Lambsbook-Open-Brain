## Title

Package Is the Contract (PIC)

## Status

ADOPTED

## Decision

The Claude Package is the complete execution contract.

Execution governance SHALL reside inside the package.

The external execution prompt SHALL act only as an execution trigger.

## Canonical Execution Trigger

Use the attached Claude Package.

Treat the package contents as the sole execution authority.

Generate the complete standalone executable Python Repository Mutation Package and the corresponding Founder Execution Package.

Continue automatically until both deliverables are complete or a single objective BLOCKED condition is established.

Do not replace the requested artifacts with Repository Truth reports, planning summaries, dependency analysis, architecture discussion, governance discussion, or recommendations.

Otherwise, return only:
1. the complete executable Python Repository Mutation Package.
