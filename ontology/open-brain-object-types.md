# Open Brain Object Types

## OBSERVATION

Raw signal.

Unverified.

Example:
Dashboard appears slow.

---

## DISCOVERY

Evidence-backed finding.

Example:
GET /api/member/me returns 403 for invited users.

---

## CERTIFIED_TRUTH

Reusable organizational truth.

Example:
UI Success Indicators Are Not Runtime Evidence.

---

## DECISION

Approved governance decision.

Example:
Events derive from Programs.

---

## LESSON

Reusable implementation learning.

Example:
Broad sed mutations create runtime instability.

---

## EVIDENCE

Supporting artifact.

Example:
runtime-log.txt
response.json
screenshot.png
