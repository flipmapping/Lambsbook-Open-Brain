TITLE:
Open Brain Semantic Object Priority Ranking

STATUS:
active

DATE:
2026-06-20

DECISION:

Open Brain semantic object implementation will proceed in the following priority order.

Tier 1

CERTIFIED_TRUTH
LESSON

Status:
Operational

Tier 2

DECISION
EVIDENCE

Reason:
Most governance workflows currently encode decisions and evidence indirectly.

These object types provide the highest leverage next expansion.

Tier 3

DISCOVERY
OBSERVATION

Reason:
Valuable but lower urgency.

Current governance and execution workflows can operate effectively without dedicated capture tooling for these classes.

RATIONALE:

The objective is to maximize decision-support capability while minimizing ontology complexity.

This prioritization aligns implementation effort with governance value.

Open Brain Architecture Review
